## ğŸ¯ Cara Menggunakan

Setiap file adalah standalone demo yang bisa di-copy paste langsung ke `lib/main.dart` untuk testing.

### Contoh:

1. Buat Flutter project baru:

   ```bash
   flutter create demo_pertemuan3
   cd demo_pertemuan3
   ```

2. Copy isi file demo (mis: `01_contact_list_demo.dart`) ke `lib/main.dart`

3. Run:
   ```bash
   flutter run
   ```
