## 🛠️ Persiapan

```bash
# 1. Install Flutter SDK dari https://flutter.dev
# 2. Verifikasi instalasi
flutter doctor

# 3. Clone repository ini
git clone https://github.com/antonprafanto/mobileSI2026.git

# 4. Masuk ke salah satu project dan run
cd contoh_proyek/pertemuan_3_catalog
flutter run
```

**Kebutuhan**: Flutter SDK · Dart SDK · Android Studio / VS Code · Git
